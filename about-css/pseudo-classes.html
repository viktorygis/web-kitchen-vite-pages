🔍 Что такое псевдоклассы?

Псевдоклассы — это специальные ключевые слова в CSS, которые добавляются к селекторам и позволяют стилизовать элементы в зависимости от их состояния или положения на странице.

  • Всегда начинаются с двоеточия : (например, :hover )
  • Позволяют стилизовать состояния, которые не заданы явно в HTML
  • Используются для реакции на действия пользователя — например, наведение мыши, клик или фокусировка

Пример

При наведении курсора на ссылку она меняет цвет:


a:hover {
  color: red; /* Меняет цвет ссылки при наведении */
}
Нажми на меня

:hover — эффект при наведении

:hover — это один из самых часто используемых псевдоклассов. Он применяется к элементу, когда пользователь наводит на него курсор мыши (или удерживает палец на мобильном устройстве).

  • Позволяет создавать интерактивные эффекты для ссылок, кнопок, картинок и других элементов.
  • Применяется моментально при наведении, без задержки.

Пример

При наведении курсора на ссылку она становится синей и подчёркнутой:


a:hover {
  color: blue;                /* Ссылка становится синей */
  text-decoration: underline; /* Добавляется подчёркивание */
}
Наведи на меня!

Когда наведёшь курсор на ссылку, она изменит цвет и станет подчёркнутой.

💡 Совет: :hover используется для стилизации любых интерактивных элементов — ссылок, кнопок, карточек и т.д.

:active — состояние нажатия

:active применяется к элементу, когда на него кликают мышью или удерживают палец на экране.
Очень часто используется для кнопок и ссылок, чтобы дать визуальный отклик при нажатии.

Пример


Кликни меня

a:active {
  color: white;
  background: #1976d2;
}
Кликни меня

Когда держишь кнопку мыши на ссылке — она изменяет цвет и фон.

💡 Совет: :active можно применять не только к ссылкам, но и к кнопкам, карточкам и любым интерактивным элементам.

:focus — когда элемент в фокусе

:focus — это псевдокласс, который применяется к элементу, когда он получает фокус. Обычно это происходит при клике мышью или при переходе к элементу с помощью клавиши Tab.

  • Особенно полезен для стилизации полей ввода, кнопок или ссылок, чтобы показать пользователю, какой элемент сейчас активен.
  • Помогает сделать формы и интерфейс более удобными и понятными.

Пример

Когда поле ввода получает фокус, появляется синяя рамка:


input:focus {
  border: 2px solid blue; /* Синяя рамка */
  outline: none;          /* Убираем стандартное выделение */
}

Кликни мышью по полю или перейди к нему с клавиатуры — и увидишь, как появляется синяя рамка.
💡 Совет: Используй :focus для выделения активных элементов и улучшения UX (пользовательского опыта) в формах!

:nth-child(n) — выбор по порядку

Псевдокласс :nth-child(n) выбирает элементы на основе их порядкового номера среди "соседей".
Особенно полезен для списков, таблиц или строк.

Выбор по порядку

Пример



    
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>



li:nth-child(2) {
  color: red; /* Второй элемент становится красным */
}
  • Первый элемент
  • Второй элемент
  • Третий элемент

Только второй элемент списка становится красным.

odd (нечётные) и even (чётные)

💡 Можно использовать odd (нечётные) и even (чётные) для стилизации сразу нескольких элементов: 


li:nth-child(odd)  { background: #f0f0f0; /* Нечётные */ }
li:nth-child(even) { background: #e0e0e0; /* Чётные */ }
  • Первый элемент
  • Второй элемент
  • Третий элемент
  • Четвёртый элемент
