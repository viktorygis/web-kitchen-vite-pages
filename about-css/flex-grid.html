Flex и Grid: Совместное использование

Flexbox и Grid — это два современных инструмента для создания макетов в CSS.
Вместе они позволяют строить гибкие, адаптивные и сложные структуры страниц.

Grid отлично подходит для построения основной структуры: например, деления страницы на колонки и строки.

Flexbox — идеален для точного выравнивания элементов внутри каждой ячейки или карточки.

Комбинируя их, ты получаешь максимальную гибкость и чистоту вёрстки!

В чем разница?

Flexbox — одномерная модель (строка или столбец). Отлично подходит для расположения элементов в линию, выравнивания и распределения пространства между ними.

Grid — двумерная модель (строки и столбцы одновременно). Применяется для построения сеток и сложных макетов.

Grid — для больших областей и структуры всей страницы.
Flexbox — для элементов внутри ячеек, карточек, кнопок и небольших групп.

Современный подход:

1️⃣ Сначала создаёшь основную сетку с помощью Grid.

2️⃣ Затем внутри каждой ячейки используешь Flexbox для выравнивания и адаптации контента.

Объединение Flex и Grid

Часто Grid используют для основной структуры, а Flex — внутри отдельных ячеек сетки.
Такой подход делает код чище, а макет — гибче.

Примеры современных сценариев:

  • Grid для основного макета, Flexbox для деталей.
    Например, карточки товаров с кнопками и текстом внутри.
  • Grid для позиционирования крупных блоков, Flexbox для адаптации.
    Например, при адаптивном дизайне меняется порядок и размер элементов внутри Grid-ячейки с помощью Flexbox.

Пример 1: Grid + Flex для списка


.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 30px;
}
.flex-list {
  display: flex;
  flex-direction: column;
  gap: 10px;
}

Итог:
Grid отвечает за общую структуру макета (расположение колонок),
Flexbox — за выравнивание и организацию элементов внутри каждой ячейки.
Такой подход делает макет гибким, аккуратным и легко читаемым. 🎨

Примеры макетов с Flex и Grid

Галерея с заголовком

В этом примере используется Grid для организации структуры галереи (заголовок + список).
Внутри Grid-ячейки применяется Flexbox для горизонтального скролла и отступов между элементами галереи. 


	.grid-gallery {
	  display: grid;
	  grid-template-rows: auto 1fr;
	  height: 250px;
	}
	.gallery-header {
	  background: #90caf9;
	  padding: 10px;
	}
	.gallery-list {
	  display: flex;
	  gap: 8px;
	  overflow-x: auto;
	  background: #e3f2fd;
	  padding: 10px;
	}

Карточки товаров

Grid используется для адаптивной сетки товаров,
а внутри каждой карточки применяется Flexbox для вертикального выравнивания содержимого:
изображения, названия и кнопки. 


	.products-grid {
	  display: grid;
	  grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
	  gap: 16px;

	.product-card {
	  display: flex;
	  flex-direction: column;
	  align-items: center;
	  padding: 14px;
	  background: #fce4ec;
	  border-radius: 8px;
	}

Советы по использованию Flex и Grid вместе

  • Используйте Grid для основной структуры страницы и размещения крупных блоков.
  • Применяйте Flexbox для выравнивания, распределения и построения содержимого внутри Grid-ячейки.
  • Не бойтесь комбинировать: например, Grid внутри Flex или наоборот.
  • Для простых одноосевых задач — Flex, для сложных сеток — Grid.
  • Проверяйте адаптивность: оба инструмента отлично подходят для responsive-дизайна.
Совет: Такой подход помогает поддерживать код чистым и масштабируемым!
