Шрифты и стили в CSS

Шрифт — это графический стиль отображения текста, определяющий внешний вид символов. Выбор шрифта влияет на восприятие информации и удобство чтения на сайте. В этом руководстве рассмотрим основные свойства, влияющие на текст, и покажем, как их применять в CSS. 


/* Пример подключения собственного шрифта */
@font-face {
  font-family: "CustomFont";
  src: url("fonts/custom-font.woff2") format("woff2"), url("fonts/custom-font.woff") format("woff");
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}

Шрифты с засечками и без засечек

Все шрифты относятся к определенному семейству , которое определяет их внешний вид и характер. Наиболее распространены два семейства:

  • Serif — шрифты с засечками: классические и элегантные, легко узнаваемы по маленьким линиям и штрихам на концах символов (например, Times New Roman).
  • Sans-serif — шрифты без засечек: современные, простые и лаконичные, имеют ровные края без дополнительных элементов (например, Arial).

Шрифты с засечками (Serif)

Times New Roman — классический шрифт с засечками
Georgia — популярный шрифт для веба

Рекомендуется для:

  • Печатных материалов
  • Длинных текстов (статьи, книги)
  • Традиционного и строгого дизайна

Шрифты без засечек (Sans-serif)

Arial — чистый и современный шрифт
Helvetica — стандарт современного дизайна

Рекомендуется для:

  • Веб-интерфейсов и экранного отображения
  • Коротких текстов (меню, кнопки, заголовки)
  • Современного и минималистичного дизайна

💡 Как задать семейство шрифта в CSS:

body {
  font-family: "Arial", sans-serif;
}
🎯 Как работает код:
  • Браузер сначала попытается использовать Arial
  • Если Arial недоступен, будет выбран любой доступный шрифт из семейства sans-serif
⚠️ Рекомендация: Всегда указывайте запасное семейство, чтобы текст отображался корректно на всех устройствах.

Совет: Для основного текста сайта обычно используют шрифты без засечек (sans-serif) — они лучше читаются на экранах. Для заголовков можно выбрать шрифты с засечками (serif) для создания визуального контраста.

Свойство font-family

font-family определяет, какие шрифты будут использоваться для отображения текста. Можно указать конкретный шрифт или семейство шрифтов (например, serif , sans-serif , monospace ). Если первый указанный шрифт недоступен, браузер выберет следующий из списка.

Рекомендуется: Всегда указывать запасные семейства, чтобы текст отображался корректно на любых устройствах. 


/* Универсальный подход для современных интерфейсов */
body {
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji",
    "Segoe UI Symbol";
}
Тип шрифта CSS-пример Вид текста Описание
Системные семейства
(serif, sans-serif, monospace) 		font-family: 'Arial', sans-serif; Пример текста Arial Безопасные веб-шрифты, есть на большинстве устройств
Google Fonts font-family: 'Roboto', sans-serif; Пример текста Roboto Популярные шрифты от Google, требуют подключения через <link> или CSS-импорт
Пользовательские шрифты font-family: 'CustomFont', fallback, sans-serif; Пример кастомного шрифта Собственные шрифты, загружаемые через @font-face

Популярные сочетания шрифтов

Шрифты без засечек (Sans-serif)

Roboto

Современный, чистый шрифт от Google. Отлично подходит для интерфейсов.

font-family: 'Roboto', sans-serif;
Open Sans

Универсальный шрифт с хорошей читаемостью. Подходит для основного текста.

font-family: 'Open Sans', sans-serif;
Montserrat

Геометрический стиль, хорошо смотрится в заголовках.

font-family: 'Montserrat', sans-serif;

Элегантный, нейтральный шрифт. Хорош для корпоративных сайтов.

font-family: 'Lato', sans-serif;
Nunito Sans

Округлый, дружелюбный стиль. Подходит для мобильных приложений и современных интерфейсов.

font-family: 'Nunito Sans', sans-serif;

Шрифты с засечками (Serif)

Roboto Slab

Современный slab-serif. Хорош для акцентных заголовков и кода.

font-family: 'Roboto Slab', serif;

Элегантный, с высоким контрастом. Отлично подходит для заголовков и модных сайтов.

font-family: 'Playfair Display', serif;

Комфортен для чтения длинного текста, оптимизирован для экранов.

font-family: 'PT Serif', serif;

Традиционный шрифт с засечками, хорошо смотрится в вебе.

font-family: 'Merriweather', serif;

Классика для веба, встроен во все системы.

font-family: Georgia, serif;

Моноширинные шрифты (Monospace)

Courier New

Стандартный моноширинный шрифт для отображения кода.

font-family: 'Courier New', monospace;
Source Code Pro

Современный моноширинный шрифт от Adobe. Отлично подходит для IDE и редакторов кода.

font-family: 'Source Code Pro', monospace;

Размер шрифта: font-size

Свойство font-size позволяет задать размер текста на странице.

Размер можно указывать в разных единицах:

  • px — пиксели, фиксированный размер.
  • em — относительная единица, зависит от размера шрифта родительского элемента.
  • rem — относительная единица, зависит от размера шрифта корневого элемента ( html ).
  • % — процент от размера шрифта родителя.
  • vw/vh — относительные единицы, основаны на ширине/высоте окна браузера.

Рекомендуется использовать rem или em для большей гибкости и доступности — так шрифт будет масштабироваться по настройкам пользователя и корневого элемента.

Примеры использования:


h1 {
  font-size: 36px; /* Крупный фиксированный заголовок */
}

p {
  font-size: 1.2rem; /* Относительный размер — зависит от размера html */
}

Совет: Используйте rem , если хотите, чтобы размеры шрифтов масштабировались относительно корневого элемента html для лучшей адаптивности.

Пример

Можно задать базовый размер для всего документа через html : 


/* Базовый размер на корневом элементе */
html {
  font-size: 16px;
}
/* Основной размер для текста */
body {
  font-size: 1rem;
}
/* Крупный заголовок */
h1 {
  font-size: 2rem;
}
/* Мелкий текст */
.small {
  font-size: 0.875rem;
}

Сравнительная таблица шрифтов

Единица Пример Видимый результат Описание
px font-size: 16px; Фиксированный размер Абсолютная единица, не зависит от настроек пользователя
em font-size: 1.2em; Относительно родителя 1em = размер шрифта родителя. Может накапливаться при вложенности
rem font-size: 1.2rem; Относительно html 1rem = размер шрифта корневого элемента. Наиболее предсказуемо и удобно для адаптивности

Рекомендуемые размеры шрифтов

12px / 0.75rem

Мелкий текст: подписи, комментарии, вспомогательная информация

16px / 1rem

Основной текст: абзацы, основной контент (стандартный размер)

20px / 1.25rem

Подзаголовки: важные элементы, но не основные заголовки

32px / 2rem

Заголовки: h2–h3, акцентные элементы страницы

font-weight

font-weight — свойство, которое управляет толщиной (насыщенностью) символов текста.
Значения бывают числовыми (от 100 до 900) и ключевыми словами ( normal , bold и др.).

Обратите внимание: Не все шрифты поддерживают полный спектр начертаний. 


/* Примеры использования */
body {
  font-weight: 400; /* Обычный текст */
}
strong {
  font-weight: 700; /* Жирный текст */
}
.light {
  font-weight: 300; /* Светлый вариант */
}
Вес CSS Видимый результат Описание
100 (Thin) font-weight: 100; Очень тонкий текст Тонкие заголовки, декоративные элементы
300 (Light) font-weight: 300; Светлый текст Мягкий, современный текст
400 (Normal) font-weight: 400; Обычный текст Стандартный вес для контента
500 (Medium) font-weight: 500; Полужирный текст Акцентные элементы, подзаголовки
700 (Bold) font-weight: 700; Жирный текст Заголовки, кнопки, важные элементы
900 (Black) font-weight: 900; Очень жирный текст Акцентные заголовки, декоративные надписи

Практические советы

  • Используйте максимум 2–3 веса на странице для визуального баланса
  • Для основного текста — normal (400) или light (300)
  • Для акцентных элементов/заголовков — medium (500) или bold (700)
  • Проверяйте доступные веса для выбранного шрифта

font-style

font-style позволяет задать стиль текста: обычный, курсивный или наклонный.
Используйте для выделения или стилизации отдельных фрагментов. 


/* Примеры */
em {
  font-style: italic;
}
.oblique-text {
  font-style: oblique 14deg; /* Наклон с углом */
}
Значение CSS Видимый результат
normal font-style: normal; Обычный текст
italic font-style: italic; Курсивный текст
oblique font-style: oblique; Наклонный текст
oblique [угол] font-style: oblique 10deg; Наклон с углом

Разница между italic и oblique

Italic

Это курсив — отдельный дизайн символов

  • Использует уникальное курсивное начертание
  • Форма букв может меняться
  • Не все шрифты содержат курсив

Oblique

Это oblique — просто наклон оригинального шрифта

  • Буквы просто наклонены, не изменяют форму
  • Поддерживается у большинства шрифтов

line-height

line-height — свойство, которое определяет высоту строки текста и влияет на вертикальные интервалы между строками.
Используйте относительные значения (например, 1.5 ) для лучшей адаптивности и читаемости. 


/* Оптимальные значения */
body {
  line-height: 1.5; /* Основной текст */
}
h1, h2, h3 {
  line-height: 1.2; /* Заголовки */
}
.compact {
  line-height: 1.2;
}
.spacious {
  line-height: 1.8;
}
Значение CSS Видимый результат Рекомендации
1.0 line-height: 1; Очень плотный текст Редко для основного текста, подходит для однострочных элементов
1.2 line-height: 1.2; Компактный текст Заголовки, короткие фрагменты
1.5 line-height: 1.5; Оптимальная читаемость Рекомендуется для основного текста и длинных абзацев
1.8 line-height: 1.8; Просторный текст Для улучшенной доступности, длинные статьи
2.0 line-height: 2; Очень большой интервал Декоративные и поэтические тексты

Примеры

Плотно

Текст с маленьким межстрочным интервалом труден для чтения, строки "слипаются".

Оптимально

Текст с хорошим межстрочным интервалом легко читается и выглядит аккуратно.

Слишком свободно

Слишком большой интервал делает текст разрозненным и теряет связанность.

text-transform

text-transform управляет регистром текста, не изменяя исходные данные.
Используется для стилизации заголовков, кнопок и других элементов интерфейса. 


/* Примеры */
.uppercase {
  text-transform: uppercase; /* ВСЕ ЗАГЛАВНЫЕ */
}
.lowercase {
  text-transform: lowercase; /* все строчные */
}
.capitalize {
  text-transform: capitalize; /* Каждое Слово С Заглавной */
}
.none {
  text-transform: none; /* Оригинальный регистр */
}
Значение CSS Видимый результат Рекомендации
uppercase text-transform: uppercase; этот текст станет заглавными буквами Подходит для кнопок, заголовков, акронимов
lowercase text-transform: lowercase; ЭТОТ ТЕКСТ СТАНЕТ СТРОЧНЫМИ Используется редко, для особых случаев
capitalize text-transform: capitalize; этот текст будет с заглавных букв Заголовки, подписи (осторожно с предлогами)
none text-transform: none; Оригинальный регистр Сброс наследуемых значений

Примеры использования

Заголовки

Section Title — верхний регистр

список последних новостей — каждое слово с заглавной

Кнопки

Особые случаи

HTML и CSS — акронимы лучше оставить без преобразования

Small Caps — альтернатива для изящного стиля

Внимание: Не используйте text-transform для замены правильного регистра в HTML. Для важных названий и имен сохраняйте корректный регистр в исходной разметке.

text-decoration

text-decoration управляет декоративными линиями для текста: подчеркивание, перечеркивание, надчеркивание.
Современный синтаксис позволяет настраивать стиль и цвет линий. 


/* Примеры */
.underline {
  text-decoration: underline;
}
.line-through {
  text-decoration: line-through;
}
.overline {
  text-decoration: overline;
}
.none {
  text-decoration: none;
}
/* Настройка стиля и цвета */
.fancy-underline {
  text-decoration: underline wavy #ff5722;
}
Значение CSS Видимый результат Рекомендации

word-spacing

word-spacing — свойство для управления интервалом между словами. Может повышать читаемость или создавать визуальные эффекты. 


/* Примеры */
.normal {
  word-spacing: normal;
}
.wide {
  word-spacing: 0.5em;
}
.narrow {
  word-spacing: -0.1em;
}
Значение CSS Видимый результат Рекомендации
normal word-spacing: normal; Стандартный интервал Рекомендуется для основного текста
0.3em word-spacing: 0.3em; Широкие интервалы Заголовки, короткие фрагменты
-0.05em word-spacing: -0.05em; Плотное расположение Узкие интерфейсы, компактные колонки

Примеры

Слишком широко

Слишком большие интервалы между словами ухудшают восприятие текста.

Оптимально

Умеренное увеличение интервала улучшает читаемость заголовков.

Плотно

Маленькие интервалы создают ощущение плотности и срочности.

letter-spacing

letter-spacing — свойство для регулировки расстояния между буквами (трекинга). Используется для улучшения читаемости и создания акцентов. 


/* Примеры */
.normal {
  letter-spacing: normal;
}
.wide {
  letter-spacing: 0.1em;
}
.tight {
  letter-spacing: -0.05em;
}
Значение CSS Видимый результат Рекомендации
normal letter-spacing: normal; Стандартное расстояние Для основного текста
0.05em letter-spacing: 0.05em; Чуть разреженные буквы Заголовки, логотипы
-0.03em letter-spacing: -0.03em; Чуть плотные буквы Кнопки, компактный текст
0.2em letter-spacing: 0.2em; Сильно разреженные буквы Декор, акценты
