Свойство position в CSS

Свойство position — это инструмент для управления расположением элементов на странице.
С помощью него можно создавать фиксированные шапки, всплывающие подсказки, модальные окна, закреплённые меню и многое другое.

Основные значения:

  • static — по умолчанию, элемент размещается в потоке документа.
  • relative — элемент перемещается относительно своего обычного положения, но остаётся в потоке.
  • absolute — элемент "вырывается" из потока и позиционируется относительно ближайшего родителя с position ( relative , absolute , fixed , sticky ).
  • fixed — элемент приклеивается к окну браузера и всегда остаётся на виду, даже при прокрутке.
  • sticky — гибрид между relative и fixed : элемент ведёт себя как обычный, пока не дойдёт до заданной позиции, а потом "прилипает".

Это позволяет делать интерфейс удобнее и приятнее для пользователя!

Абсолютное позиционирование ( position: absolute; )

position: absolute; — как волшебство ✨: элемент становится независимым от других и располагается там, где укажете, относительно ближайшего родителя с position (не равным static ).

Это удобно для всплывающих подсказок, кнопок, элементов интерфейса, которые должны быть строго на своём месте.

Пример:

<div class="parent">
  <div class="child">Абсолютный элемент</div>
</div>
Абсолютный элемент
  • .child позиционируется относительно .parent , у которого position: relative; .
  • Отступы top: 20px и left: 30px задают точное положение внутри родителя.
  • Элемент «вырывается» из обычного потока и не влияет на расположение других элементов.

Частые шаблоны позиционирования

Сместить на 50% ширины родителя (по X)

.parent {
  position: relative;
}
.child  {
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
По центру по X

left: 50% ставит левый край ребёнка в центр родителя, translateX(-50%) сдвигает обратно на половину ширины ребёнка → идеальное центрирование по X.

По центру по X и Y

.child {
  position: absolute;
  left: 50%; top: 50%;
  transform: translate(-50%, -50%);
}
Полный центр

Классический способ центрировать элемент посередине контейнера.

Без transform (если известна ширина)

.child {
  position: absolute;
  width: 150px;
  left: 50%;
  margin-left: -75px; /* - width/2 */
}
Без transform

Подходит для фиксированной ширины. Для адаптивных блоков лучше использовать transform .

Центр без координат (inset + auto)

.child {
  position: absolute;
  inset: 0;           /* top/right/bottom/left: 0 */
  margin: auto;       /* центрирует по обеим осям */
  width: 140px; height: 60px;
}
inset:0 + auto

Работает при заданных размерах ребёнка.

Смещение на 50% + доп. отступ

.child {
  position: absolute;
  left: 50%;
  transform: translateX(calc(-50% + 20px));
}
Центр + 20px вправо

Удобно, когда нужна точка опоры в центре, но сдвинутая на фиксированную величину.

Углы и растяжение

/* Нижний‑правый угол */
.child { position: absolute; right: 12px; bottom: 12px; }

/* Растянуть внутри родителя */
.child { position: absolute; inset: 10px 16px; }
Правый низ
Растянуто (inset)

Используйте top/right/bottom/left или сокращение inset для нужной геометрии.

Памятка

  • Абсолютно позиционированный элемент ищет ближайшего предка с position: relative|absolute|fixed|sticky и позиционируется относительно него.
  • Если такого предка нет — позиционирование идёт относительно вьюпорта/страницы.
  • Для горизонтального центрирования чаще всего: left:50% + translateX(-50%) .
  • Для полного центрирования: left:50%; top:50%; transform: translate(-50%,-50%);

Фиксированное позиционирование ( position: fixed; )

position: fixed; — элемент буквально "приклеен" к своему месту на экране 🚀 и всегда виден, даже если страницу прокручивают.
Это отличный выбор для кнопки "Наверх", всплывающего меню, уведомлений и панелей управления.

Пример


<div class="fixed-box">Я всегда на виду!</div>
ищи меня в правом верхнем углу
Я всегда на виду!
  • Элемент фиксируется относительно окна браузера, а не родителя.
  • Отступы top и right задают его положение на экране.
  • Даже если прокрутить страницу, элемент останется на своём месте.
  • Подходит для элементов интерфейса, которые должны быть всегда доступны.

z-index: управление слоями

Свойство z-index управляет порядком наложения элементов на странице: чем больше значение, тем выше элемент находится над другими.
Работает только для элементов с position отличным от static .

  • z-index может быть положительным, отрицательным или равным нулю.
  • Влияет на "плоскость" внутри одного контекстa наложения ( stacking context ).
  • Удобен для модальных окон, всплывающих подсказок, выпадающих меню.
z-index:2
z-index:1

.box1 {
  position: absolute;
  left: 50px;
  top: 20px;
  z-index: 2;
}
.box2 {
  position: absolute;
  left: 90px;
  top: 36px;
  z-index: 1;
}
Важно: z-index работает только для элементов с position: relative , absolute , fixed или sticky .

Примеры всех видов position

static
По умолчанию
Элемент занимает место в потоке, не реагирует на top/left/right/bottom.
static

.element {
  position: static;
}
relative
Смещение относительно себя
Сохраняет место в потоке, но можно сдвинуть элемент с помощью top/left/right/bottom.
relative

.element {
  position: relative;
  top: 8px;
  left: 18px;
}
absolute
Абсолютное позиционирование
Выпадает из потока, позиционируется относительно родителя с position или body.
absolute
контейнер
Родитель position: relative; 

.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 16px;
  right: 20px;
}
fixed
Фиксировано на экране
Позиционируется относительно окна браузера, не зависит от скролла.
fixed

.element {
  position: fixed;
  bottom: 12px;
  right: 18px;
}
sticky
Липкое позиционирование
Остаётся на месте, пока не достигнет порога, затем фиксируется.
sticky (top: 0)

.element {
  position: sticky;
  top: 0;
}

Советы и лайфхаки по position

  • absolute ищет ближайшего родителя с position , иначе позиционируется относительно <body> .
  • У sticky должен быть родитель с overflow: visible , иначе эффект не сработает.
  • Для z-index нужен position , отличный от static .
  • Свойства top , left , bottom , right действуют только для relative , absolute , fixed , sticky .
  • Для удобства абсолютного позиционирования используйте position: relative на родительском элементе.
  • Не злоупотребляйте fixed — они всегда будут поверх основного контента.
Совет:

Используйте position: sticky для закрепления меню, сносок, заголовков таблиц и прочего. Это удобно и кроссбраузерно!
