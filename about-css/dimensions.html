Блочная модель и box-sizing

Блочная модель CSS — это схема, по которой рассчитываются размеры любого элемента. В неё входят:
контент (content), внутренние отступы (padding), границы (border) и внешние отступы (margin).
box-sizing — свойство, определяющее, как браузер вычисляет ширину и высоту элемента: учитывает ли только контент или также padding и border.

Контент Padding Border Margin
Значение Описание Формула расчета ширины
content-box По умолчанию.
width и height включают только контент, padding и border добавляются сверху. 		Общая ширина = width + padding + border
border-box width и height включают контент, padding и border. Общая ширина = width (всё внутри)

Сравнение content-box и border-box

content-box (по умолчанию)

div {
  width: 500px;
  padding: 20px;
  border: 5px solid black;
}

Общая ширина = 500px (content) + 20px × 2 (padding) + 5px × 2 (border) = 550px
box-sizing: content-box;

border-box

div {
  width: 500px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: border-box;
}

Общая ширина = 500px (content + padding + border)
box-sizing: border-box;

Интерактивная демонстрация box-sizing:

Фактическая ширина: 200 px

width: 200 px + padding: 40 px (20px × 2) + border: 30 px (15px × 2) = 270 px

Совет: всегда используйте box-sizing: border-box

Для современной верстки практически всегда лучше использовать box-sizing: border-box для всех элементов.
Это избавит вас от неожиданных переполнений, сложных расчетов и проблем с адаптивностью. 

*, *::before, *::after {
  box-sizing: border-box;
}
  • Блоки легко размещать и делать адаптивными
  • Размеры элементов всегда предсказуемы
  • Верстка становится проще и стабильнее
💡 Важно помнить:
  • content-box — считает только содержимое, но добавляет отступы и границы сверху.
  • border-box — включает в ширину и высоту всё: содержимое, отступы и границы.

Управление размерами элементов в CSS

В CSS ты можешь гибко управлять размерами с помощью различных свойств:

  • width и height — задают ширину и высоту содержимого.
    Если используется box-sizing: border-box , ширина и высота включают padding и border.
  • max-width и max-height — ограничивают максимальную ширину и высоту элемента.
    Элемент никогда не будет больше этого значения, даже если родитель больше.
  • min-width и min-height — задают минимальные размеры.
    Элемент не уменьшится меньше указанного значения.

Пример

div {
  width: 50%;        /* Половина ширины родителя */
  max-width: 300px;  /* Не больше 300px */
  min-width: 100px;  /* Не меньше 100px */
}
Этот блок всегда будет от 100px до 300px шириной и займёт 50% ширины родителя, если позволяет пространство.
