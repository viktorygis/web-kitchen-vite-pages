Свойство font-size в CSS

Свойство font-size определяет размер шрифта элемента.
Это одно из наиболее важных свойств CSS, влияющее на читаемость текста и визуальную иерархию страницы.

Обзор единиц измерения в CSS

В CSS доступны фиксированные и относительные единицы — выбирайте их в зависимости от задачи!

  • px — фиксированный пиксель, для точных размеров
  • % — процент от родителя, для гибких макетов
  • em — от размера шрифта родителя, для масштабируемых элементов
  • rem — от размера корневого элемента, для единого масштаба
  • vw/vh — от размера окна, для адаптивных блоков
  • pt/cm/mm — для печатных стилей 
div { width: 100vw; height: 50vh; }

Баннер на всю ширину и половину высоты экрана

px — пиксели

Пиксели — фиксированная единица, не зависит от устройства. Используйте для точных размеров.

  • Гарантирует одинаковый размер на всех экранах
  • Не адаптируется под размер родителя или viewport
  • Подходит для границ, иконок, кнопок

Пример

Текст всегда будет 16 пикселей

p { font-size: 16px; }

Текст всегда будет 16 пикселей

% — проценты

Проценты — относительная единица, зависит от размера родителя.

  • Гибкие блоки, легко адаптируются
  • Значение 50% — половина ширины или высоты родителя
  • Используйте для адаптивных сеток, контейнеров
div { width: 50%; }

Блок займет половину ширины родителя

em и rem — относительные единицы

em зависит от размера шрифта родителя.
rem — всегда от размера корня ( <html> ).

  • em удобно для вложенных элементов
  • rem — для единого масштаба на всей странице
<html style="font-size: 16px;">
  <div style="font-size: 20px;">
    <p style="font-size: 2em;">Текст с em</p>     
    <p style="font-size: 2rem;">Текст с rem</p>   
  </div>
</html>

Текст с em (2em = 40px)
Текст с rem (2rem = 32px)

em — от родителя, rem — от корня

vw / vh — размеры окна

vw — 1% ширины окна
vh — 1% высоты окна

  • Используйте для баннеров, заставок, модальных окон
  • Позволяют создавать элементы, которые занимают часть экрана

Пример

div { width: 100vw; height: 50vh; }
Баннер занимает всю ширину и половину высоты экрана

pt, cm, mm — для печати

Абсолютные единицы — используются для печатных стилей, где важны реальные размеры (дюймы, миллиметры, пункты).

  • Точный контроль над размером для печати
  • Не адаптируются под экран
  • Редко применяются для веб-интерфейса
p { font-size: 12pt; } /* пункты */
p { font-size: 1cm; }  /* сантиметры */
p { font-size: 5mm; }  /* миллиметры */

Рекомендации по выбору единицы

  • Для адаптивности: Используйте %, em, rem, vw, vh
  • Для точности: px, pt
  • Для типографики: rem — для базового размера, em — для вложенных элементов
  • Для доступности: Избегайте px для основного текста
  • Для печати: pt, cm, mm

Визуальный пример: баннер

div {
  width: 100vw;
  height: 50vh;
  background-color: lightblue;
}
