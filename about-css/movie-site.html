Разбор страницы: что реализовано

  1. <!DOCTYPE html> — объявление типа документа, сообщает браузеру, что используется HTML5.
  2. <html lang="en"> — корневой элемент, язык страницы — английский.
  3. <head>
    • <meta charset="utf-8"> — кодировка страницы UTF-8.
    • <meta name="viewport"> — для адаптивности, корректное отображение на мобильных.
    • <title> — заголовок страницы во вкладке браузера.
    • <style>...</style> — встроенные CSS-стили для оформления страницы.
  4. <body>
    • <header> — "шапка" сайта, содержит лого и иконку меню.
      Стили: position: sticky; top: 0; z-index: 100; display: flex; align-items: center; background; border-bottom — закрепляет хедер сверху при скролле, делает его флекс-контейнером для выравнивания.
    • <img> — логотип сайта и иконка меню (картинки) в шапке и футере, а также постер фильма.
    • <div class="category"> — блок-категории фильмов с тегами жанров.
      Стили: display: flex; overflow: auto; ... — горизонтальный скроллируемый список жанров, выделение активного жанра.
    • <div class="films"> — грид-контейнер для фильмов.
      Стили: display: grid; grid-template-columns: repeat(2, 1fr); ... — сетка для карточек фильмов, отступы и оформление.
    • <div class="film"> — отдельная карточка фильма.
      Стили: position: relative; ... — для позиционирования рейтинга.
    • <div class="title"> — название фильма.
    • <div class="rating"> — рейтинг фильма, расположен поверх постера.
      Стили: position: absolute; top: 10px; right: 10px; ...
    • <footer> — подвал сайта, содержит повтор логотипа, документы, контакты и права.
      Стили: margin-top, margin-bottom, padding, text-align: center, opacity — оформление футера и его элементов.
    • <ul>, <li> — используются для списков (документы, контакты).
    • <a> — ссылки на страницы документов и контакты.

Использованные стили (CSS):

  • code: padding, border-radius, font-family, background-color, color — оформление фрагментов кода.
  • body: margin, padding, background, color, font-family — сброс отступов, цвет, шрифт.
  • ul: list-style: none; padding: 0; — убирает маркеры и отступы у списков.
  • a: text-decoration: none; color: #fff; — ссылки без подчеркивания, белые.
  • .container: width: 100%; height: 100%; — контейнер на всю ширину и высоту.
  • header: position: sticky; ... — закрепление и оформление шапки, флекс-выравнивание, фон.
  • .logo: height: 24px; — высота логотипа.
  • .menu-logo: width: 32px; height: 32px; — размеры иконки меню (гамбургер).
  • .category: display: flex; overflow: auto; ... — оформление и горизонтальная прокрутка жанров.
  • .tag: border-radius, background, line-height, ... — стилизация жанров, активный жанр выделяется цветом.
  • footer: margin, padding, text-align, opacity, ... — стилизация футера, центрирование, полупрозрачность текста.
  • .films: display: grid; grid-template-columns: repeat(2, 1fr); ... — сетка для карточек фильмов.
  • .film: position: relative; — для позиционирования рейтинга.
  • .film img: width: 100%; border-radius: 16px; — оформление обложки фильма.
  • .film .title: font-size, font-weight, margin-top — стилизация названия фильма.
  • .film .rating: position: absolute; top: 10px; right: 10px; ... — оформление рейтинга поверх постера.
  • footer .list-documents, .contact, .rights: margin, text-align, opacity — оформление списков и информации в футере.

Пошагово что реализовано:

  • Создан адаптивный HTML-шаблон для страницы каталога фильмов.
  • Реализована "липкая" (sticky) шапка с логотипом и иконкой меню.
  • Добавлен блок категорий (жанров фильмов) с горизонтальной прокруткой и выделением активного жанра.
  • Реализована сетка фильмов: карточка фильма содержит постер, название и рейтинг поверх изображения.
  • Добавлен современный подвал с логотипом, списком документов, контактами и информацией о правах.
  • Внедрены стили для улучшения визуального восприятия: скругления, тени, полупрозрачные элементы, сетка, стилизация ссылок и списков.
  • Адаптивная верстка, оформление для различных разрешений экрана.
