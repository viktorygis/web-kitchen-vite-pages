Стилизация кнопок в HTML и CSS

Кнопки — важная часть интерфейса.
С помощью CSS можно сделать их яркими, современными, удобными: изменить цвет, форму, добавить эффекты наведения, тени или иконки.

Основные свойства для стилизации кнопок

  • background-color — цвет фона кнопки
  • color — цвет текста
  • border — рамка кнопки (или её отсутствие)
  • padding — внутренние отступы, пространство между текстом и краями кнопки
  • border-radius — закругление углов кнопки
  • cursor — вид курсора (например, pointer для указателя)

Пример базовой стилизации кнопки


	<button>Нажми меня</button>

	<style>
	button {
	background-color: #4CAF50; /* Зелёный фон */
	color: white;              /* Белый текст */
	border: none;              /* Без рамки */
	padding: 10px 20px;        /* Внутренние отступы */
	border-radius: 5px;        /* Закруглённые углы */
	cursor: pointer;           /* Курсор-указатель */
	}
	</style>

Кнопка выглядит аккуратно, современно и буквально приглашает к действию! 🖱️💚

Интерактивность: псевдокласс :hover

Чтобы кнопка "оживала" при наведении, используют псевдокласс :hover .

Пример кнопки с эффектом наведения


	<button>Наведи на меня</button>

	<style>
	button {
	background-color: #008CBA; /* Синий фон */
	color: white;              /* Белый текст */
	border: none;
	padding: 10px 20px;
	border-radius: 5px;
	cursor: pointer;
	}

	button:hover {
	background-color: #005f73; /* Фон становится темнее */
	}
	</style>

При наведении фон кнопки становится темнее — она выглядит "живой" и привлекательной! 🖱️✨

Тени для кнопок: box-shadow

Свойство box-shadow добавляет тень к кнопке, делая её объёмнее и выразительнее.

Пример кнопки с тенью


	<button>Кнопка с тенью</button>

	<style>
	button {
	background-color: #4CAF50; /* Зелёный фон */
	color: white;              /* Белый текст */
	padding: 10px 20px;
	border-radius: 8px;        /* Закруглённые углы */
	border: none;
	box-shadow: 3px 3px 10px rgba(0, 0, 0, 0.3); /* Лёгкая тень */
	cursor: pointer;
	}
	</style>
Как работает box-shadow :
  • 3px 3px — смещение тени вправо и вниз
  • 10px — размытие (чем больше число, тем мягче тень)
  • rgba(0, 0, 0, 0.3) — цвет тени, 30% прозрачности
Тень делает кнопку "воздушной" и современной!

Современные стили кнопок — с описанием

Primary — стандартная цветная кнопка для главных действий.
Outline — прозрачная кнопка с цветной рамкой.
Success — зелёная кнопка для успешных или подтверждающих действий.
Danger — красная кнопка для удаления и предупреждений.
Градиент — модный плавный переход цветов.
Стекло — лёгкая, с прозрачностью и размытием.
Тень — белая с эффектом парящей тени.
Тёмная — минималистичная тёмная кнопка.
Ссылка — выглядит как текстовая ссылка.
Icon — круглая кнопка с иконкой, например “лайк”.

✨ Экспериментируйте с цветами, тенями, градиентами, скруглением и иконками, чтобы создавать собственные современные стили кнопок!
