Введение в HTML

HTML (HyperText Markup Language) — это основной язык разметки для создания веб-страниц и веб-приложений.

HTML используется для структурирования контента на веб-странице, включая текст, изображения, ссылки, таблицы, формы и другие элементы. Он определяет, как эти элементы будут отображаться в браузере.

HTML работает в сочетании с CSS (Cascading Style Sheets) для стилизации и оформления страниц, а также с JavaScript для добавления интерактивности и динамического поведения.

История HTML началась в начале 1990-х годов, когда Тим Бернерс-Ли разработал первую версию языка для обмена научными документами. С тех пор HTML прошёл несколько этапов развития, и текущая версия HTML5 стала стандартом для создания современных веб-приложений.

HTML является основой веб-разработки, и знание его основ необходимо для создания и понимания веб-контента.

Структура HTML-документа

HTML-документ имеет иерархическую структуру , состоящую из элементов, которые могут содержать текст, изображения, ссылки и другие медиа. Элементы HTML организованы в виде дерева, где каждый элемент может иметь вложенные элементы. Это позволяет создавать сложные и структурированные веб-страницы.

  • HTML описывает структуру страницы с помощью специальных элементов — тегов .
  • Каждый тег заключён в угловые скобки <...> и сообщает браузеру, как отображать содержимое.
  • HTML-документ начинается с <!DOCTYPE html> и состоит из корневого элемента <html> , внутри которого находятся разделы <head> и <body> .
  • Внутри <body> размещается всё видимое содержимое страницы. 
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Моя первая страница</title>
  </head>
  <body>
    <h1>Заголовок страницы</h1>
    <p>Это первый абзац.</p>
  </body>
</html>

Примеры базовой HTML-разметки

HTML — основа вашего сайта. Освоив базовые теги и структуру, вы сможете создавать простые и сложные страницы, добавлять изображения, формировать списки, создавать таблицы и формы. 

<h1>Главный заголовок</h1>
<p>Абзац текста.</p>
<img src="image.jpg" alt="Описание">
<a href="https://example.com">Ссылка</a>

Теги HTML

В HTML существует множество тегов — для заголовков, абзацев, изображений, списков, ссылок, таблиц, форм, аудио, видео и других элементов.
Основная задача HTML — описать смысл и структуру контента, не отвечая за внешний вид (стилизация выполняется с помощью CSS).

Теги HTML можно разделить на несколько категорий в зависимости от их назначения:

Структурные теги

Эти теги образуют основу HTML-документа и используются для создания структуры страницы.

  • <div> - контейнер для других элементов
  • <p> - абзац текста
  • <img> - изображение
  • <a> - ссылка
Теги форматирования

Эти теги используются для изменения внешнего вида текста и других элементов.

  • <strong> - выделение важного текста
  • <em> - выделение текста курсивом
  • <u> - подчеркивание текста
  • <del> - зачеркивание текста
Мультимедиа теги

Эти теги используются для вставки мультимедийного контента на веб-страницу.

  • <audio> - аудиофайл
  • <video> - видеоплеер
  • <iframe> - встроенный фрейм
Теги форм

Эти теги используются для создания интерактивных форм на веб-странице.

  • <form> - контейнер для формы
  • <input> - поле ввода
  • <textarea> - многострочное текстовое поле
  • <button> - кнопка
Семантические теги

Эти теги используются для определения структуры и значимости контента на веб-странице.

  • <header> - заголовок страницы
  • <nav> - навигационное меню
  • <main> - основное содержимое страницы
  • <footer> - подвал страницы
Другие теги

Эти теги используются для создания различных элементов на веб-странице.

  • <div> - блочный контейнер
  • <span> - строчный контейнер
  • <blockquote> - цитата
