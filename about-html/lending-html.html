Форматирование

Жирный текст используется для выделения важных частей предложения, например: Это очень важно! с помощью тега <b> .
Иногда курсивный текст (с помощью тега <i> ) помогает подчеркнуть мысли, например: Обратите внимание на детали.
Подчеркнутый текст (с помощью тега <u> ) часто применяется для выделения ссылок или специальных терминов, например: это термин .

Можно комбинировать стили, например: Жирный, курсив и подчеркнутый одновременно (с помощью тегов <b> , <i> и <u> ) — для особо важных примечаний.
Маркером выделяют (с помощью тега <mark> ) ключевые моменты,
а мелкий текст (с помощью тега <small> ) — для сносок или пояснений.
Зачёркнутый текст (с помощью тега <del> ) обозначает устаревшую или отменённую информацию.

При написании формул удобно использовать индексы: H 2 O (с помощью тега <sub> ),
а для обозначения степеней — верхний индекс, например: x 2 (с помощью тега <sup> ).
Для примеров кода используют тег <code> .

Таким образом, грамотное использование тегов форматирования (с помощью тега <b> ) позволяет сделать текст более наглядным (с помощью тега <i> ), структурированным (с помощью тега <u> ) и удобным для чтения (с помощью тега <mark> ).

Теги

Основные теги

  • <h1> - Заголовок первого уровня
  • <h2> - Заголовок второго уровня
  • <h3> - Заголовок третьего уровня
  • <h4> - Заголовок четвертого уровня
  • <p> - Абзац текста

Базовые теги

  • <html> - Корневой элемент HTML-документа
  • <head> - Заголовок документа
  • <body> - Основное содержимое страницы
  • <title> - Заголовок страницы
  • <meta> - Метаданные страницы

Теги форматирования

  • <b> - Жирный текст
  • <i> - Курсивный текст
  • <u> - Подчеркнутый текст
  • <mark> - Выделенный текст
  • <small> - Маленький текст

Теги стилей

  • <link> - Стили для внешних ресурсов
  • <style> - Стили для внутреннего использования

Семантические теги

  • <header> - Заголовок страницы
  • <nav> - Навигационное меню
  • <main> - Основной контент
  • <footer> - Подвал страницы
  • <article> - Статья
  • <section> - Секция
  • <aside> - Боковая панель
  • <h1> - <h6> - Заголовки

Мета-теги и SEO

  • <title> - Заголовок страницы
  • <meta name="description"> - Описание страницы
  • <meta name="robots"> - Управление индексацией страницы
  • <meta property="og:title"> - Заголовок для соцсетей

Теги html

Теги HTML - это строительные блоки веб-страниц. Каждый тег имеет свое назначение и используется для разметки контента.

Семантика

Семантика HTML - это смысл, который стоит за тегами. Правильное использование семантических тегов помогает улучшить доступность и SEO.

HTML5

HTML5 - это последняя версия HTML, которая включает новые элементы и атрибуты, улучшая поддержку мультимедиа и графики.
