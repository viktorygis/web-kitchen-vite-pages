Теги HTML

Теги HTML — это элементы, используемые для разметки содержимого веб-страницы.

  • Теги HTML представляют собой команды, заключенные в угловые скобки, которые сообщают браузеру, как отображать содержимое.
    Например, тег <p> используется для создания параграфов текста.
  • Теги могут быть одиночными, как <br> (перенос строки), или парными, как <div></div> (блоковый элемент).
Теги HTML могут быть разделены на несколько категорий:
- базовые теги,
- семантические теги,
- теги стилей
- теги форматирования текста.
Каждая категория имеет свои особенности и предназначение.

Базовые теги HTML

Базовые теги HTML — это строительные блоки веб-страницы.

Виды базовых тегов:

  • Структурные теги: <html> , <head> , <body>
  • Теги для текста: <p> , <h1>–<h6> , <blockquote>
  • Теги для списков: <ul> , <ol> , <li>
  • Теги для ссылок и медиа: <a> , <img> , <video>

Пример базового HTML-документа:

<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Пример документа</title>
    </head>
    <body>
        <h1>Заголовок первого уровня</h1>
        <p>Параграф текста.</p>
        <ul>
            <li>Элемент списка</li>
        </ul>
    </body>
</html>

Семантические теги

Главные теги для структуры HTML-документа.

Семантические теги HTML — это теги, которые имеют четкое значение и предназначение.

Виды семантических тегов:

  • <header> — шапка страницы или раздела.
  • <nav> — навигация сайта.
  • <main> — основной контент (один на странице).
  • <section> — логический раздел, обычно с заголовком.
  • <article> — самостоятельная публикация.
  • <footer> — подвал страницы или раздела.

Пример семантических HTML-документа:

<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Пример семантического документа</title>
    </head>
    <body>
        <header>
            <h1>Заголовок первого уровня</h1>
        </header>
        <nav>
            <ul>
                <li><a href="#html-basics">Основы HTML</a></li>
                <li><a href="#html-semantic">Семантические теги</a></li>
                <li><a href="#html-style">Теги стилей</a></li>
            </ul>
        </nav>
        <main>
            <section id="html-basics">
                <h2>Основы HTML</h2>
                <p>Введение в HTML и его структуру.</p>
            </section>
            <section id="html-semantic">
                <h2>Семантические теги</h2>
                <p>Описание семантических тегов и их использование.</p>
            </section>
            <section id="html-style">
                <h2>Теги стилей</h2>
                <p>Как подключать CSS-стили к HTML-документу.</p>
            </section>
        </main>
        <footer>
            <p>© 2023 Пример документа</p>
        </footer>
    </body>
</html>

Теги стилей

Теги стилей HTML — это теги, которые используются для определения стилей и оформления элементов на веб-странице.

Виды тегов стилей:

  • <style> — встроенные стили в HTML.
  • <link> — подключение внешних CSS-файлов.

Пример тегов стилей- подключение внешних CSS-файлов:

<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Пример семантического документа</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>

    </body>
</html>

Пример тегов стилей- встроенные стили в HTML:

<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Пример семантического документа</title>
        <style>
            body {
                font-family: Arial, sans-serif;
                background-color: #f4f4f4;
            }
            h1 {
                color: #333;
            }
        </style>
    </head>
    <body>

    </body>
</html>

Мультимедийные теги

Теги мультимедийные HTML — это теги, которые используются для добавления медиа-контента на веб-страницу.

Виды мультимедийных тегов:

  • <video> — для вставки видео.
  • <audio> — для вставки аудио.
  • <img> — для вставки изображений.
  • <iframe> — для встраивания других веб-страниц.
  • <object> — для вставки объектов, таких как Flash или PDF.
  • <embed> — для встраивания медиа-контента.
  • <canvas> — для рисования графики с помощью JavaScript.
