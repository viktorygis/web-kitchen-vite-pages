Программная навигация во Vue Router

Программная навигация — это способ менять маршруты не кликами по ссылкам, а прямо из кода.
Это полезно для редиректов после логина, выхода, отправки формы и других действий.

Примеры использования:

  • Редирект после успешного входа
  • Перенаправление после выхода из аккаунта
  • Переход к странице успеха после отправки формы

Для этого используются методы $router.push() , $router.replace() и $router.go() вместо <router-link> .

Переход на страницу с помощью $router.push()

$router.push() позволяет программно отправлять пользователя на другой маршрут.
Маршрут добавляется в историю браузера!

Для чего использовать push:

  • После успешного логина
  • После регистрации
  • После отправки формы
  • Для автоматических редиректов
this.$router.push("/dashboard");
  • this — текущий экземпляр Vue
  • $router — объект маршрутизатора
  • push() — метод перехода

Можно использовать и в шаблоне:

<button @click="$router.push('/profile')">Перейти в профиль</button>

Можно передавать параметры, если нужен динамический маршрут: 

this.$router.push({ name: "profile", params: { id: 42 } });

name должен соответствовать имени маршрута в router.js .

Замена маршрута с помощью $router.replace()

$router.replace() заменяет текущий маршрут без добавления его в историю.
Назад вернуться нельзя!

Для чего использовать replace:

  • После выхода из аккаунта
  • После отправки формы
  • Для автоматических редиректов
this.$router.replace("/login");
  • Используйте после выхода, чтобы нельзя было вернуться на защищённую страницу
  • Используйте после отправки формы, чтобы не отправить её повторно
  • Используйте при автоматических редиректах
Метод Что делает? Можно вернуться назад?
$router.push() Добавляет новый маршрут Да
$router.replace() Заменяет текущий маршрут Нет

Программная кнопка "Назад" с помощью $router.go()

$router.go() позволяет перемещаться по истории браузера (назад/вперёд) программно. 

this.$router.go(-1); // Назад
this.$router.go(1);  // Вперёд
this.$router.go(-2); // На 2 шага назад

Пример кнопки:

<button @click="$router.go(-1)">🔙 Вернуться назад</button>

Полезно: для многошаговых форм, возврата из вложенных страниц, если не знаешь точный маршрут.

Практика: программная навигация

  • Сделай кнопку, которая переводит пользователя в "Личный кабинет" по клику.
  • Сделай кнопку "Назад", которая перемещает на 2 страницы назад.
  • Сделай кнопку "Выйти", которая заменяет текущий маршрут на /login.

App.vue

<!-- App.vue -->
<template>
  <div>
    <router-link to="/profile">Личный кабинет</router-link>
    <router-link to="/login">Вход</router-link>
    <router-view/>
  </div>
</template>

Пример компонента с программной навигацией

<!-- NavigationDemo.vue -->
<template>
  <div>
    <button @click="goToProfile">Перейти в личный кабинет</button>
    <button @click="goBack">🔙 Назад (на 2 страницы)</button>
    <button @click="logout">Выйти</button>
  </div>
</template>

<script>
export default {
  methods: {
    goToProfile() {
      this.$router.push("/profile");
    },
    goBack() {
      this.$router.go(-2);
    },
    logout() {
      this.$router.replace("/login");
    }
  }
}
</script>

Частые ошибки и вопросы

  • Ошибка: this.router.push(...) — забыли $ перед router!
  • Ошибка: используешь push() , а нужен replace() — пользователь может вернуться назад.
  • Вопрос: Как передать параметры?
    this.$router.push({ name: "profile", params: { id: 42 } })
  • Вопрос: Как сделать редирект после логина/выхода/отправки? — Просто вызывай нужный метод в нужный момент!
