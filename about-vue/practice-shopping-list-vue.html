🛍 Мой список покупок

  • {{ item }}

Описание проекта

Список покупок — это простое веб-приложение на Vue.js для создания и управления списком товаров. Пользователь может быстро вносить покупки, удалять их из списка, а интерфейс мгновенно реагирует на все изменения. Код легко расширять и модифицировать под свои задачи.

  • Добавление товаров через поле ввода и кнопку "Добавить" (или клавишу Enter);
  • Удаление товаров из списка при помощи кнопки "Удалить" рядом с каждым пунктом;
  • Мгновенное обновление интерфейса — реактивность реализована благодаря возможностям Vue 3.

Пример кода HTML

<div id="app">

<!-- Заголовок -->
    <h2>🛍 Мой список покупок</h2>

<!-- Поле ввода и кнопка добавления -->
    <input type="text" v-model="newItem" placeholder="Введите товар..." @keyup.enter="addItem" />

    <!-- Кнопка добавления -->
    <button @click="addItem">Добавить</button>

<!-- Список покупок -->
    <ul>
    <!-- Динамический вывод списка с возможностью удаления -->
        <li v-for="(item, index) in items" :key="index">
            {{ item }}
            <button @click="removeItem(index)">Удалить</button>
        </li>
    </ul>
</div>

Описание кода

  • v-model="newItem" — двусторонняя привязка для поля ввода нового товара;
  • @keyup.enter="addItem" — добавление товара по нажатию Enter;
  • @click="addItem" — добавление товара по клику на кнопку;
  • v-for="(item, index) in items" — динамический вывод списка покупок;
  • @click="removeItem(index)" — удаление товара по индексу при клике на кнопку "Удалить".

Пример кода JavaScript

const app = Vue.createApp({
    data() {
        return {
            newItem: "",
            items: [],
        };
    },
    methods: {
        addItem() {
            if (this.newItem.trim() !== "") {
                this.items.push(this.newItem.trim());
                this.newItem = "";
            }
        },
        removeItem(index) {
            this.items.splice(index, 1);
        },
    },
});
app.mount("#app");

Описание кода

  • data() — состояние приложения: введённый текст (newItem) и массив товаров (items);
  • addItem() — добавляет новый товар (если поле не пустое);
  • removeItem(index) — удаляет товар по индексу, вызывается кнопкой "Удалить" напротив товара;
  • @keyup.enter="addItem" и @click="addItem" — обратчики событий для быстрого добавления товаров;
  • v-model="newItem" — двусторонняя привязка с полем ввода;
  • v-for="(item, index) in items" — динамический вывод списка покупок в реальном времени.

Преимущества

  • Проект прост для понимания, но демонстрирует самые ключевые концепции Vue;
  • Легко доработать: добавить сохранение списка, стили, отметки купленного и так далее;
  • Подходит как для самостоятельной тренировки, так и для использования на практике.
