v-model

v-model — директива Vue для двустороннего связывания данных и формы.

С помощью v-model вы можете легко синхронизировать данные и элементы формы:

Для чего используется:

  • Автоматическое обновление переменных при вводе в текстовые поля, то есть <input type="text">
  • Связывание состояния чекбоксов то есть <input type="checkbox">
  • Привязка выбранной опции в группе радио-кнопок то есть <input type="radio">
  • Привязка выбранных значаний в выпадающих списках к данным select то есть <select>
Использование v-model значительно упрощает работу с формами во Vue!

Текстовые поля и v-model

v-model для текстовых полей - автоматически связывает значение поля с переменной. С помощью этого вы можете легко получать и обновлять текстовые данные в вашем приложении Vue.

❌ Без v-model нужно вручную обрабатывать событие @input:
<input type="text" @input="message = $event.target.value">

✅ А теперь гораздо проще:
<input type="text" v-model="message">

Интерактивный пример:

Попробуйте ввести текст в поле ниже:

Привет, {{ name }}!

Пример кода:

<div id="app">
  <input type="text" v-model="name">
  <p>Привет, {{ name }}!</p>
</div>

<script>
  Vue.createApp({
    data() {
      return { name: "" }
    }
  }).mount("#app");
</script>
Объяснение кода:
  • v-model связывает поле ввода с переменной name в data()
  • При вводе текста в поле, значение name автоматически обновляется
  • Вывод {{ name }} мгновенно показывает текущее значение
🚀 Результат: при вводе текста в поле, переменная name обновляется автоматически, и приветствие меняется мгновенно!

Чекбоксы и v-model

Чекбоксы позволяют пользователям выбирать или снимать выбор с опций. С помощью v-model вы можете легко связать состояние чекбокса с булевой переменной в вашем приложении Vue.

v-model для чекбоксов - автоматически обновляет переменную в зависимости от состояния чекбокса (отмечен/не отмечен).

Интерактивный пример:

Попробуйте отметить или снять отметку с чекбокса ниже:

Я согласен с условиями

Согласие: {{ isChecked }}

Пример кода:

<input type="checkbox" v-model="isChecked">
<p>Согласие: {{ isChecked }}</p>

data() {
  return { isChecked: false }
}
Объяснение кода:
  • v-model связывает чекбокс с переменной isChecked в data()
  • Когда чекбокс отмечен, isChecked становится true, иначе false
  • Вывод {{ isChecked }} показывает текущее состояние чекбокса
🚀 Результат: при отмечании или снятии отметки с чекбокса, переменная isChecked обновляется автоматически, и состояние чекбокса отображается мгновенно!

Радио-кнопки и v-model

Радио-кнопки позволяют пользователям выбирать одну опцию из группы. С помощью v-model вы можете легко связать выбранную опцию с переменной в вашем приложении Vue.

Интерактивный пример:

Попробуйте выбрать одну из опций ниже:

Опция 1
Опция 2

Выбрано: {{ selectedOption }}

Пример кода:

<input type="radio" v-model="selectedOption" value="option1"> Опция 1
<input type="radio" v-model="selectedOption" value="option2"> Опция 2
<p>Выбрано: {{ selectedOption }}</p>

data() {
  return { selectedOption: "" }
}
Объяснение кода:
  • v-model связывает группу радио-кнопок с переменной selectedOption в data()
  • При выборе опции, selectedOption обновляется значением выбранной радио-кнопки
  • Вывод {{ selectedOption }} показывает текущий выбор пользователя
🚀 Результат: при выборе одной из радио-кнопок, переменная selectedOption обновляется автоматически, и текущий выбор отображается мгновенно!

Выпадающие списки и v-model

Выпадающие списки позволяют пользователям выбирать одну или несколько опций из списка. С помощью v-model вы можете легко связать выбранные опции с переменной в вашем приложении Vue.

Интерактивный пример:

Попробуйте выбрать страну из списка ниже:

Выбрана страна: {{ selectedCountry }}

Пример кода:

<select v-model="selectedCountry">
  <option value="russia">Россия</option>
  <option value="usa">США</option>
  <option value="germany">Германия</option>
</select>
<p>Выбрана страна: {{ selectedCountry }}</p>

data() {
  return { selectedCountry: "" }
}

Сравнение v-model с @input

Рассмотрим разницу между использованием @input и v-model для двусторонней привязки данных в Vue.

@input - это событие, которое срабатывает при вводе данных в поле. При использовании @input вам нужно вручную обновлять переменную, используя $event.target.value . Вот пример:

<input type="text" @input="message = $event.target.value">
data() {
  return { message: "" }
}

С другой стороны, v-model автоматически связывает значение поля с переменной. Вот как это выглядит:

<input type="text" v-model="message">
data() {
  return { message: "" }
}

Основные отличия:

  • v-model упрощает код, устраняя необходимость в ручном обновлении переменной.
  • С @input вы должны явно указать, как обновлять переменную, что может привести к ошибкам.
  • v-model обеспечивает более чистый и понятный синтаксис для двусторонней привязки данных.
Вывод: Используйте v-model для двусторонней привязки данных, так как это проще и менее подвержено ошибкам по сравнению с @input .

Советы и типичные ошибки

  • Для двухсторонней привязки всегда используйте v-model
  • Переменные для v-model должны быть в data()
  • Для множественного выбора используйте v-model с массивом
  • Следите за типом переменной: чекбоксы — Boolean , select/radio — String
Частая ошибка: переменная для v-model отсутствует в data() — привязка не работает!

Практика: мини-форма с v-model


<div id="app">
  <input type="text" v-model="username" placeholder="Имя">
  <input type="checkbox" v-model="agreed"> Согласен с правилами
  <select v-model="country">
    <option value="russia">Россия</option>
    <option value="usa">США</option>
    <option value="germany">Германия</option>
  </select>

  <p>Имя: {{ username }}</p>
  <p>Согласие: {{ agreed }}</p>
  <p>Страна: {{ country }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return {
      username: "",
      agreed: false,
      country: ""
    }
  }
}).mount("#app");
</script>
В этом примере: v-model мгновенно синхронизирует данные формы и переменные!

Реальные примеры использования v-model во Vue

Пример 1. Живая фильтрация списка

В этом примере мы будем фильтровать список пользователей по имени.

  • {{ user }}

<input type="text" v-model="search" placeholder="Поиск пользователя">
<ul>
  <li v-for="user in filteredUsers" :key="user">{{ user }}</li>
</ul>
Пояснение: v-model связывает поле ввода с переменной search , а список фильтруется на лету.

Пример 2. Управление настройками профиля

В этом примере мы будем управлять настройками профиля пользователя с помощью v-model.

{{ profile }} 
<input type="text" v-model="profile.name">
<input type="email" v-model="profile.email">
<input type="checkbox" v-model="profile.notifications">
Пояснение: v-model работает с вложенными объектами, все изменения сразу видны.

Пример 3. Множественный выбор в select

В этом примере мыбудем выбирать несколько языков программирования.

Выбрано: {{ selected }}

 <select v-model="selected" multiple>
  <option value="js">JavaScript</option>
  <option value="vue">Vue</option>
  <option value="react">React</option>
</select>
Пояснение: v-model поддерживает массивы для множественного выбора!
