methods, computed и watch во Vue

methods — обычные функции для обработки событий.
computed — вычисляемые свойства, зависящие от данных и кешируемые.
watch — наблюдатели за изменением данных для выполнения побочных эффектов.
Все три используются внутри Vue-компонентов для управления логикой приложения.

methods

Определяются в объекте methods . Обычно используются для обработки событий (например, @click ). Не кешируются.

  • Обычные функции
  • Используются для обработки событий
  • Не кешируются
methods: {
   myMethod() {
      // код метода
   }
}
computed

Определяются в объекте computed . Пересчитываются только при изменении зависимостей (например, data() ) и кешируются.

  • Геттеры (и сеттеры)
  • Используются для вычислений на основе данных
  • Кешируются
computed: {
 myComputed() {
  // код вычисляемого свойства
 }
}
watch

Определяются в объекте watch . Следят за изменением переменной и запускают функцию, например, при изменении data() .

  • Функции-обработчики
  • Используются для асинхронных действий и побочных эффектов
  • Не кешируются
watch: {
 myVar(newValue, oldValue) {
  // код обработчика
 }
}

Методы в Vue

methods — это функции, определённые внутри объекта methods в компоненте Vue. Они обычно используются для обработки событий, таких как клики по кнопкам, изменения ввода и другие взаимодействия пользователя.

Где используются методы:

  • Обработчики событий (например, @click, @input)
  • Логика, связанная с действиями пользователя
  • Манипуляции с данными компонента

Как использовать методы:

Методы определяются в объекте methods компонента Vue. Они могут быть вызваны из шаблона с помощью директивы @click, @input и других.

Пример 1:

Создадим простой счётчик с кнопкой для увеличения значения.

Счётчик: {{ count }}

<div id="counter-demo">
  <p>Счётчик: {{ count }}</p>
  <button @click="increaseCount">Увеличить</button>
</div>

<script>
Vue.createApp({
  data() {
    return { count: 0 }
  },
  methods: {
    increaseCount() {
      this.count++;
    }
  }
}).mount("#counter-demo");
</script>
Объяснение кода:
  • data() возвращает объект с переменной count, инициализированной нулём.
  • В объекте methods определён метод increaseCount, который увеличивает значение count на 1.
  • В шаблоне кнопка привязана к методу через директиву @click.

Пример 2:

Расширенный счётчик с увеличением, уменьшением и сбросом значения.

Счётчик: {{ count }}

<div id="counter-full">
  <p>Счётчик: {{ count }}</p>
  <button @click="increase">+</button>
  <button @click="decrease">-</button>
  <button @click="reset">Сброс</button>
</div>

<script>
Vue.createApp({
  data() { return { count: 0 } },
  methods: {
    increase() { this.count++; },
    decrease() { this.count--; },
    reset() { this.count = 0; }
  }
}).mount("#counter-full");
</script>
Объяснение кода:
  • Метод increase увеличивает счётчик на 1.
  • Метод decrease уменьшает счётчик на 1.
  • Метод reset сбрасывает счётчик в 0.

Передача параметров и работа с this

Методы могут принимать параметры, которые передаются при вызове из шаблона. Внутри методов this ссылается на экземпляр Vue-компонента, позволяя получать доступ к данным и другим методам.

Пример 1:

Создадим счётчик с кнопками, увеличивающими значение на разное количество.

Счётчик: {{ count }}

<div id="counter-step">
  <p>Счётчик: {{ count }}</p>
  <button @click="increase(1)">+1</button>
  <button @click="increase(5)">+5</button>
  <button @click="increase(10)">+10</button>
</div>

<script>
Vue.createApp({
  data() { return { count: 0 } },
  methods: {
    increase(value) { this.count += value }
  }
}).mount("#counter-step");
</script>
Объяснение кода:
  • Метод increase принимает параметр value, который указывает, на сколько увеличить счётчик.
  • В шаблоне кнопки вызывают метод с разными аргументами: 1, 5 и 10.
  • Внутри метода используется this.count для доступа к переменной счётчика и её обновления.

Пример 2:

Создадим метод для форматирования имени с заглавной буквы.

Имя: {{ formatName(name) }}

<div id="name-demo">
  <input v-model="name" placeholder="Введите имя">
  <p>Имя: {{ formatName(name) }}</p>
</div>

<script>
Vue.createApp({
  data() { return { name: "алиса" } },
  methods: {
    formatName(str) {
      return str.charAt(0).toUpperCase() + str.slice(1);
    }
  }
}).mount("#name-demo");
</script>
Объяснение кода:
  • Метод formatName принимает строку str и возвращает её с заглавной первой буквой.
  • В шаблоне метод вызывается с текущим значением из поля ввода name.

Пример 3:

Создадим метод для преобразования текста в нижний регистр.

Результат: {{ toLowerCase(text) }}

<div id="lowercase-demo">
  <input v-model="text" placeholder="Введите текст">
  <p>Результат: {{ toLowerCase(text) }}</p>
</div>

<script>
Vue.createApp({
  data() { return { text: "" } },
  methods: {
    toLowerCase(str) { return str.toLowerCase(); }
  }
}).mount("#lowercase-demo");
</script>
Объяснение кода:
  • Метод toLowerCase принимает строку str и возвращает её в нижнем регистре с помощью встроенного метода JavaScript toLowerCase().
  • В шаблоне метод вызывается с текущим значением из поля ввода text.

computed: вычисляемые свойства

computed определяются в объекте computed. Пересчитываются только при изменении зависимостей (например, data()) и кешируются.

Что нужно знать о computed:

  • Используются для вычислений на основе данных компонента.
  • Пересчитываются только при изменении зависимостей.
  • Кешируются для оптимизации производительности.

В отличие от методов, которые выполняются при каждом вызове, вычисляемые свойства кешируются до тех пор, пока не изменятся их зависимости. Это делает их идеальными для сложных вычислений, которые не нужно выполнять каждый раз при рендеринге.

Пример:

Создадим вычисляемое свойство, которое возвращает удвоенное значение счётчика.

Исходное значение: {{ computedCount }}

Удвоенное (computed): {{ doubleCount }}

<div id="computed-demo">
  <h4>Исходное значение: {{ computedCount }}</h4>
  <h4>Удвоенное (computed): {{ doubleCount }}</h4>
  <button @click="computedCount++">Увеличить</button>
</div>

<script>
Vue.createApp({
  data() { return { computedCount: 5 } },
  computed: {
    doubleCount() { return this.computedCount * 2 }
  }
}).mount("#computed-demo");
</script>
Объяснение кода:
  • В объекте data() определена переменная computedCount, инициализированная значением 5.
  • В объекте computed определено вычисляемое свойство doubleCount, которое возвращает удвоенное значение computedCount.
  • В шаблоне отображаются оба значения, и кнопка увеличивает computedCount на 1 при каждом клике.

🚀 computed идеально подходят для производительных вычислений, которые зависят от реактивных данных.

watch: наблюдатель

watch определяются в объекте watch. Следят за изменением переменной и запускают функцию, например, при изменении data().

Что нужно знать о watch:

  • Используются для асинхронных действий и побочных эффектов.
  • Следят за изменением реактивных данных.
  • Не кешируются.

В отличие от computed , которые предназначены для вычислений на основе данных, наблюдатели watch используются для выполнения действий в ответ на изменения данных, таких как запросы к API или обновление других частей состояния.

Пример:

Создадим наблюдатель, который будет логировать изменения счётчика в консоль.

Счётчик: {{ scoring }}

<div id="watch-demo">
  <h4>Счётчик: {{ scoring }}</h4>
  <button @click="scoring++">Увеличить</button>
</div>
<script>
Vue.createApp({
  data() { return { scoring: 0 } },
  watch: {
    scoring(newValue, oldValue) {
      console.log(`Счётчик изменился с ${oldValue} на ${newValue}`);
    }
  }
}).mount("#watch-demo");
</script>
Объяснение кода:
  • В объекте data() определена переменная scoring, инициализированная значением 0.
  • В объекте watch определён наблюдатель для переменной scoring, который логирует в консоль старое и новое значение при каждом изменении.
  • В шаблоне отображается текущее значение счётчика, и кнопка увеличивает его на 1 при каждом клике.

🚀 watch подходит для асинхронных действий и побочных эффектов.
Например, запрос к API при изменении значения.

Практика

Создание простого калькулятора с computed и watch

Создадим простой калькулятор, который будет использовать computed для вычисления удвоенного, утроенного значения и квадратного корня из введённого числа. Также добавим watch , чтобы сбрасывать значение, если оно становится отрицательным.

Удвоенное: {{ double }}

Утроенное: {{ triple }}

Квадратный корень: {{ sqrt }}

Число отрицательное!

<div id="calc-demo">
  <input v-model.number="num" type="number" placeholder="Введите число">
  <p>Удвоенное: {{ double }}</p>
  <p>Утроенное: {{ triple }}</p>
  <p>Квадратный корень: {{ sqrt }}</p>
  <p v-if="num < 0">Число отрицательное!</p>
</div>

<script>
Vue.createApp({
  data() { return { num: 0 } },
  computed: {
    double() { return this.num * 2 },
    triple() { return this.num * 3 },
    sqrt() { return Math.sqrt(Math.abs(this.num)) }
  },
  watch: {
    num(newVal) {
      if (newVal < 0) {
        this.num = 0;
      }
    }
  }
}).mount("#calc-demo");
</script>
Объяснение кода:
  • В объекте data() определена переменная num, инициализированная значением 0.
  • В объекте computed определены три вычисляемых свойства: double, triple и sqrt, которые возвращают удвоенное, утроенное значение и квадратный корень из абсолютного значения num соответственно.
  • В объекте watch определён наблюдатель для переменной num, который сбрасывает значение на 0, если оно становится отрицательным.
  • В шаблоне отображаются результаты вычислений, а также предупреждение, если введённое число отрицательное.

Реальные примеры

Реальные примеры использования методов, computed и watch во Vue для решения различных задач.

Пример 1. Список с обработкой строк

Создадим метод для форматирования строк в списке. Здесь импользуется метод для преобразования первой буквы каждого элемента списка в заглавную.

  • {{ capitalize(item) }}
<ul>
  <li v-for="item in items" :key="item">{{ capitalize(item) }}</li>
</ul>

<script>
Vue.createApp({
  data() {
    return {
      items: ["яблоко", "банан", "вишня"]
    };
  },
  methods: {
    capitalize(str) {
      return str.charAt(0).toUpperCase() + str.slice(1);
    }
  }
}).mount("#str-demo");
</script>
Объяснение кода:
  • Метод capitalize принимает строку и возвращает её с заглавной первой буквой.
  • В шаблоне метод вызывается для каждого элемента списка items, форматируя его перед отображением.

Пример 2. Фильтр с computed

Создадим вычисляемое свойство для фильтрации списка на основе поискового запроса.

  • {{ item }}
<div id="filter-demo">
  <input v-model="search" placeholder="Поиск">
  <ul>
    <li v-for="item in filtered" :key="item">{{ item }}</li>
  </ul>
</div>
<script>
Vue.createApp({
  data() {
    return {
      search: "",
      items: ["яблоко", "банан", "вишня", "апельсин", "груша"]
    };
  },
  computed: {
    filtered() {
      return this.items.filter(item =>
        item.toLowerCase().includes(this.search.toLowerCase())
      );
    }
  }
}).mount("#filter-demo");
</script>
Объяснение кода:
  • Вычисляемое свойство filtered возвращает отфильтрованный список элементов, содержащих строку из переменной search.
  • В шаблоне отображается только те элементы, которые соответствуют поисковому запросу.

Пример 3. Watch для контроля состояния

Создадим наблюдатель, который будет сбрасывать значение баллов, если оно становится отрицательным.

Баллы: {{ score }}


<div id="watch-example">
  <input v-model.number="score" type="number" placeholder="Баллы">
  <p>Баллы: {{ score }}</p>
</div>
<script>
Vue.createApp({
  data() {
    return {
      score: 0
    };
  },
  watch: {
    score(newVal) {
      if (newVal < 0) {
        this.score = 0;
      }
    }
  }
}).mount("#watch-example");
</script>
Объяснение кода:
  • Наблюдатель для переменной score сбрасывает её значение на 0, если оно становится отрицательным.
  • В шаблоне отображается текущее значение баллов, и пользователь может вводить число через поле ввода.
