Реактивность в компонентах Vue

Реактивность в компонентах Vue - это способность автоматически обновлять интерфейс при изменении данных. Vue предоставляет два основных инструмента для работы с реактивностью: computed и watch.

Основные понятия:

  • computed — свойства, которые зависят от других данных и автоматически обновляются при их изменении. Идеальны для вычислений и форматирования.
  • watch — позволяет выполнять код при изменении значения переменной. Полезно для асинхронных задач, валидаций и API-запросов.
  • Оба инструмента помогают создавать динамичные и отзывчивые интерфейсы, улучшая пользовательский опыт.

Вычисляемые свойства computed

computed - это свойства, которые автоматически пересчитываются при изменении зависимых данных.

Как использовать computed:

  • Определяются в объекте computed внутри компонента Vue
  • имяСвойства() — функция, возвращающая вычисленное значение
  • return значение; — возвращаемое значение свойства
computed: {
  имяСвойства() {
    return значение;
  }
}

Пример использования:

В этом примере мы создаём вычисляемое свойство greeting, которое зависит от переменной name. Когда пользователь вводит имя, приветствие автоматически обновляется.


<div id="computed-demo">
  <input v-model="name" placeholder="Введите имя">
  <p>👋 Привет, {{ greeting }}!</p>
</div>

<script>
Vue.createApp({
  data() {
    return { name: "" }
  },
  computed: {
    greeting() {
      return this.name ? `друг ${this.name}` : "незнакомец";
    }
  }
}).mount("#computed-demo");
</script>
Объяснение кода:
  • Переменная name хранит введённое пользователем имя.
  • Вычисляемое свойство greeting возвращает строку приветствия на основе значения name.
  • Когда пользователь вводит имя, Vue автоматически обновляет отображаемое приветствие.
Особенности:
  • Если зависимые данные не изменились, значение берётся из кэша
  • Используйте computed , когда результат зависит от других данных и должен обновляться автоматически

Упражнение: Сделайте computed, возвращающее текст в верхнем регистре


<div id="computed-uppercase-demo">
  <input v-model="text" placeholder="Введите текст">
  <p>🔠 В верхнем регистре: {{ upperText }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return { text: "" }
  },
  computed: {
    upperText() {
      return this.text.toUpperCase();
    }
  }
}).mount("#computed-uppercase-demo");
</script>

Наблюдатели watch

watch — это способ выполнить код при изменении значения переменной. 

watch: {
  имя_переменной(новоеЗначение, староеЗначение) {
    // Код при изменении
  }
}
                            

<div id="watch-demo">
  <input v-model="name" placeholder="Введите имя">
  <p>🕵️ Проверка: {{ message }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return {
      name: "",
      message: "Ожидаю ввода..."
    }
  },
  watch: {
    name(newVal) {
      if (newVal.length < 3) {
        this.message = "Слишком короткое имя! ❌";
      } else {
        this.message = "Отлично! ✅";
      }
    }
  }
}).mount("#watch-demo");
</script>
Особенности:
  • watch всегда выполняет код при изменении значения
  • Идеально для асинхронных задач, валидаций, API-запросов

Упражнение: watch для чётности числа


<div id="watch-even-demo">
  <p>🔢 Число: {{ count }}</p>
  <button @click="count++">+1</button>
  <p>🧐 {{ message }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return {
      count: 0,
      message: "Начнём считать! ⏳"
    }
  },
  watch: {
    count(newValue) {
      newValue % 2 === 0
        ? this.message = "Число чётное! ✅"
        : this.message = "Число нечётное! 🔴";
    }
  }
}).mount("#watch-even-demo");
</script>

Сравнение computed и watch

Когда что использовать?
  • computed — когда нужно вычислить новое значение на основе других данных, результат можно кэшировать
  • watch — когда нужно выполнить код при изменении (например, асинхронный запрос или валидацию) 

<div id="comparison-demo">
  <input v-model="name" placeholder="Введите имя">
  <p>📝 Приветствие (computed): {{ greeting }}</p>
  <p>🕵️ Проверка (watch): {{ message }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return {
      name: "",
      message: "Ожидаю ввода..."
    }
  },
  computed: {
    greeting() {
      return this.name ? `Привет, ${this.name}!` : "Привет, незнакомец!";
    }
  },
  watch: {
    name(newValue) {
      if (newValue.length < 3) {
        this.message = "Имя слишком короткое ❌";
      } else {
        this.message = "Имя подходит! ✅";
      }
    }
  }
}).mount("#comparison-demo");
</script>
Кратко:
  • computed — просто вычислить значение (приветствие, форматирование, суммы и т.д.)
  • watch — запустить код при изменении (валидация, API-запрос, асинхронщина)

Асинхронные задачи: отложенный поиск с watch

watch отлично подходит для асинхронных и отложенных действий. 

<div id="async-demo">
  <input v-model="query" placeholder="🔍 Введите запрос">
  <p>📊 Результат: {{ result }}</p>
</div>

<script>
Vue.createApp({
  data() {
    return {
      query: "",
      result: "Нет данных..."
    }
  },
  watch: {
    query(newValue) {
      this.result = "⏳ Ищем...";
      setTimeout(() => {
        this.result = `🔍 Найдено: ${newValue}`;
      }, 1000);
    }
  }
}).mount("#async-demo");
</script>
Где пригодится?
  • Отложенный поиск (debounce)
  • Автоматическая загрузка данных
  • Фильтрация и обновление списков
