Динамические компоненты во Vue

Динамические компоненты - это способ менять отображаемый компонент на лету, без сложной логики. Вместо громоздких v-if или v-show, ты можешь просто указать, какой компонент должен быть показан в данный момент. Это особенно полезно для создания вкладок, модальных окон и других интерфейсных элементов, где нужно быстро менять содержимое.

Основной инструмент:

  • Тег <component :is="componentName"></component> позволяет "вставлять" нужный компонент в зависимости от значения переменной componentName.
  • Значение componentName может быть строкой с именем зарегистрированного компонента или самим объектом компонента.
  • Можно передавать props и слушать события, как в обычных компонентах.

Базовое переключение между компонентами

Базовое переключение между компонентами - простой пример использования динамических компонентов для переключения между двумя компонентами по нажатию кнопок. Это демонстрирует, как легко можно менять отображаемый компонент без сложной логики.

Как это работает:

  • Две кнопки изменяют значение переменной currentComponent .
  • Тег <component :is="currentComponent"> отображает компонент, имя которого хранится в currentComponent .
  • Vue автоматически обновляет отображение при изменении currentComponent .

Пример:

<div id="app">
  <button @click="currentComponent = 'first-component'">Показать первый</button>
  <button @click="currentComponent = 'second-component'">Показать второй</button>
  <component :is="currentComponent"></component>
</div>

<script>
Vue.createApp({
  data() {
    return { currentComponent: "first-component" }
  },
  components: {
    "first-component": { template: "<p>📌 Это первый компонент!</p>" },
    "second-component": { template: "<p>🔥 А это второй компонент!</p>" }
  }
}).mount("#app");
</script>
Объяснение кода:
  • Создаём две кнопки, которые меняют currentComponent при клике.
  • Используем <component :is="currentComponent"> для отображения нужного компонента.
  • Регистрируем два простых компонента с шаблонами.

🚀 Вывод: Динамические компоненты во Vue позволяют легко переключаться между разными компонентами на основе состояния приложения, делая код чище и проще для понимания. Это мощный инструмент для создания интерактивных интерфейсов. Попробуй изменить пример и добавить свои компоненты!

Передача props в динамический компонент

Передача props в динамический компонент - это способ передать данные в компоненты, которые выбираются динамически с помощью тега <component>. Это позволяет создавать более гибкие и переиспользуемые компоненты, которые могут принимать различные входные данные в зависимости от контекста.

Как это работает:

  • Используем атрибут :is для выбора компонента.
  • Передаём props так же, как в обычных компонентах.
  • Компоненты принимают props и используют их в своих шаблонах.

Пример:

<div id="app">
  <button @click="currentComponent = 'hello-component'">Привет</button>
  <button @click="currentComponent = 'bye-component'">Пока</button>
  <component :is="currentComponent" :message="customMessage"></component>
</div>

<script>
Vue.createApp({
  data() {
    return {
      currentComponent: "hello-component",
      customMessage: "Передано через prop!"
    }
  },
  components: {
    "hello-component": {
      props: ["message"],
      template: `<p>👋 {{ message }}</p>`
    },
    "bye-component": {
      props: ["message"],
      template: `<p>👋 Пока! {{ message }}</p>`
    }
  }
}).mount("#app");
</script>

Объяснение кода:

  • Две кнопки меняют значение currentComponent для переключения между компонентами.
  • Тег <component :is="currentComponent" :message="customMessage"> передаёт prop message в выбранный компонент.
  • Оба компонента принимают prop message и отображают его в своём шаблоне.

🚀 Вывод: Передача props в динамический компонент позволяет гибко управлять данными, передаваемыми в компоненты, которые выбираются динамически. Это делает код более читаемым и переиспользуемым.

Пример: система вкладок (Tabs)

Система вкладок - это интерфейсный элемент, который позволяет пользователям переключаться между разными разделами контента, не покидая текущую страницу. Во Vue.js динамические компоненты идеально подходят для реализации такой системы, так как они позволяют легко менять отображаемый контент на основе выбранной вкладки.

Пример:

<div id="app">
  <button @click="activeTab = 'tab-1'">📝 Описание</button>
  <button @click="activeTab = 'tab-2'">📊 Характеристики</button>
  <button @click="activeTab = 'tab-3'">🗣 Отзывы</button>
  <component :is="activeTab"></component>
</div>

<script>
Vue.createApp({
  data() {
    return { activeTab: "tab-1" }
  },
  components: {
    "tab-1": { template: "<p>📝 Это описание продукта.</p>" },
    "tab-2": { template: "<p>📊 Здесь характеристики.</p>" },
    "tab-3": { template: "<p>🗣 Здесь будут отзывы клиентов.</p>" }
  }
}).mount("#app");
</script>

Объяснение кода:

  • Три кнопки изменяют значение activeTab для переключения между вкладками.
  • Тег <component :is="activeTab"> отображает компонент, соответствующий выбранной вкладке.
  • Каждая вкладка зарегистрирована как отдельный компонент с собственным шаблоном.
🚀 Вывод: Динамические компоненты во Vue позволяют легко создавать системы вкладок, делая интерфейс более интерактивным и удобным для пользователей. Попробуй расширить пример, добавив больше вкладок или изменив их содержимое!

Вопрос 1:
Какой текст увидит пользователь сразу после загрузки страницы? 
<div id="app">
  <button @click="currentComponent = 'big-cat'">Показать Большого кота</button>
  <button @click="currentComponent = 'tiny-dog'">Показать Маленькую собаку</button>
  <component :is="currentComponent" :message="'Лови привет!'"></component>
</div>

<script>
Vue.createApp({
  data() {
    return { currentComponent: "big-cat" }; // По умолчанию показывается "big-cat"
  },
  components: {
    "big-cat": {
      template: "<h2>🐱 {{ message }}</h2>",
      props: ["message"]
    },
    "tiny-dog": {
      template: "<p>🐶 {{ message }}</p>",
      props: ["message"]
    }
  }
}).mount("#app");
</script>
Ответ: 🐱 Лови привет!
