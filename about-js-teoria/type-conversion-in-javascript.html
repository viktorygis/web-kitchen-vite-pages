Преобразование типов в JavaScript

Преобразование типов — автоматическое или явное изменение типа значения.
JavaScript использует неявное преобразование в операциях и предоставляет функции для явного преобразования.

Основные виды преобразований:

  • Строковое ( String() )
  • Численное ( Number() )
  • Логическое ( Boolean() )

Когда происходит:

  • При математических операциях
  • При сравнении значений
  • При выводе данных
  • В логических контекстах

Строковое преобразование

String(value) — преобразует значение в строку.
Происходит автоматически при конкатенации строк или выводе.

Синтаксис:

String(value)

Где value — любое значение для преобразования.

Примеры:

String(true)    // "true"
String(null)     // "null"
String(123)      // "123"
String(undefined) // "undefined"

Особенности:

  • Работает для всех типов данных
  • Для объектов вызывает метод toString()
  • Для массивов возвращает элементы через запятую

Практическое применение:

  • Формирование строковых сообщений
  • Подготовка данных для вывода
  • Конкатенация с другими строками

Численное преобразование

Number(value) — преобразует значение в число.
Происходит автоматически в математических операциях.

Синтаксис:

Number(value)

Где value — любое значение для преобразования.

Примеры:

Number("123")   // 123
Number("123z")  // NaN
Number(true)    // 1
Number(false)   // 0

Правила преобразования:

Значение Результат
undefined NaN
null 0
true / false 1 / 0
Пустая строка 0
Пробельные символы Обрезаются, затем преобразуется
Нечисловая строка NaN

Альтернативные методы:

parseInt("100px") // 100 (целое число)
parseFloat("12.5em") // 12.5 (дробное число)
+"123" // 123 (унарный плюс)

Логическое преобразование

Boolean(value) — преобразует значение в логический тип.
Происходит автоматически в условиях и логических операциях.

Синтаксис:

Boolean(value)

Где value — любое значение для преобразования.

Примеры:

Boolean(1)       // true
Boolean(0)       // false
Boolean("text")  // true
Boolean("")      // false

Правила преобразования:

Значение Результат
0 , "" , null , undefined , NaN false
Любые другие значения true
Строка "0" true (не пустая)
Пробелы в строке true (не пустая)

Особенности:

  • Все объекты (включая пустые) преобразуются в true
  • Массивы (даже пустые) — true
  • Функции — true

Сводная таблица преобразований

Значение String() Number() Boolean()
undefined "undefined" NaN false
null "null" 0 false
true "true" 1 true
false "false" 0 false
"" (пустая строка) "" 0 false
"123" "123" 123 true
"123z" "123z" NaN true
0 "0" 0 false
1 "1" 1 true

Практические советы:

  1. Используйте явное преобразование для ясности кода
  2. Будьте осторожны с неявным преобразованием при сравнениях ( == )
  3. Для проверки на NaN используйте isNaN()
  4. Для чисел предпочитайте Number() или унарный + перед parseInt/parseFloat
