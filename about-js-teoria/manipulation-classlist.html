Манипуляция классами элементов в DOM

classList — это свойство, позволяющее удобно управлять CSS-классами любого DOM-элемента.
С его помощью можно добавлять, удалять, переключать и проверять наличие классов без перезаписи всей строки.

  • Добавляй классы: делай элементы ярче, добавляй стили и эффекты.
  • Убирай классы: возвращай элементы к исходному виду.
  • Переключай классы: быстро меняй стили по клику или действию пользователя.

Основы работы с classList

classList — это объект, предоставляющий методы для работы с классами элемента.

Пример HTML:

<div id="box" class="highlight">Это блок</div>

Получить список классов элемента:

const box = document.getElementById("box");
console.log(box.classList); // DOMTokenList ["highlight"]

Методы classList

Добавление класса add()
box.classList.add("shadow");
Добавляет указанный класс к элементу, если его нет.
Удаление класса remove()
box.classList.remove("highlight");
Удаляет указанный класс из элемента.
Проверка класса contains()
box.classList.contains("shadow");
Проверяет, есть ли класс у элемента (true/false).
Переключение класса toggle()
box.classList.toggle("toggle");
Добавляет класс, если его нет, и удаляет, если есть.

add - Добавление класса

Метод add добавляет указанный класс к элементу, если его нет.

element.classList.add("название_класса"); 


<div id="add-box">Я изменюсь!</div>

<script>
document.getElementById("add-box").classList.add("highlight");
// Теперь у div будет класс highlight
</script>

Я изменюсь! (classList.add)

remove - Удаление класса

Метод remove удаляет указанный класс из элемента.

element.classList.remove("название_класса"); 


<div id="remove-box" class="highlight">Я потеряю стиль!</div>

<script>
document.getElementById("remove-box").classList.remove("highlight");
// Класс highlight будет удалён
</script>
Я не изменюсь!
Я потеряю стиль! (classList.remove)

toggle - Переключение класса

Метод toggle добавляет класс, если его нет, и удаляет, если он есть.

element.classList.toggle("название_класса"); 


<div id="toggle-box">Жми на меня!</div>

<script>
document.getElementById("toggle-box").addEventListener("click", () => {
    document.getElementById("toggle-box").classList.toggle("toggle");
});
// Класс toggle будет добавлен/удалён по клику
</script>
Я не изменюсь!
Жми на меня! (classList.toggle)

contains - Проверка наличия класса

Метод contains возвращает true , если элемент содержит указанный класс, иначе false .

element.classList.contains("название_класса"); 


<div id="contains-box" class="highlight">Проверим класс!</div>

<script>
const hasHighlight = document.getElementById("contains-box").classList.contains("highlight");
console.log(hasHighlight); // true
</script>
Я не изменюсь!
Проверим класс! (classList.contains)

Советы по использованию

Совет: Используй toggle для простого переключения классов — это короче и удобнее, чем add и remove вместе.

Практика: Манипуляция классами

Попробуй добавить, удалить и переключить класс

Это интерактивный блок

Пример кода:

const box = document.getElementById("demo-box");
// Добавить класс
box.classList.add("shadow");
// Удалить класс
box.classList.remove("shadow");
// Проверить наличие класса
box.classList.contains("highlight"); // true/false
// Переключить класс
box.classList.toggle("toggle");

FAQ: Вопросы по classList

  • Можно ли добавить сразу несколько классов?
    Да, например: box.classList.add("one", "two");
  • Как удалить сразу несколько классов?
    Аналогично: box.classList.remove("one", "two");
  • Можно ли узнать весь список классов элемента?
    Да, через box.classList или box.className
  • Как работает toggle с вторым аргументом?
    box.classList.toggle("toggle", true); — добавляет класс.
    box.classList.toggle("toggle", false); — удаляет класс.
  • Как проверить наличие класса перед удалением?
    Используй contains — это предотвращает ошибки.
